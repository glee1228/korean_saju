# Changelog

## 0.1.0 — 2026-05-14

Initial release. Ported from a precision-validated Dart implementation.

### Features

- 4기둥(四柱) 사주 계산 — 연주·월주·일주·시주
- **야자시(夜子時) / 조자시(朝子時) 분리법 토글** (`yaja_si_separated=True` 기본)
- 진태양시(眞太陽時) 출생지 경도 보정
- 입춘(立春) 절입 기준 연주, 절기 절입 기준 월주
- KASI(한국천문연구원) 음력 1900–2050 + 절기 2000–2027 데이터 임베드
- 범위 밖 절기는 Meeus 천문 계산기로 자동 fallback (정확도 ±15분)
- 십신(十神), 십이운성(十二運星), 십이신살(十二神煞), 신살(神煞)
- 격국(格局), 외격(外格), 용신(用神), 일간 강약, 조후(調候), 합충(合沖)
- 대운(大運), 세운(歲運)
- 60갑자 공망(空亡, 旬空) 계산
